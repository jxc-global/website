# JXC Website

The public website for JXC.

JXC builds and operates AI agent teams for real business workflows.

Live site: https://jxc.global/

## About

This site presents JXC's work in a minimal black-and-white visual system: direct copy, sharp edges, and no unnecessary decoration.

## Built With

- Astro
- HTML and CSS
- SVG favicon

## Development

Install dependencies:

```bash
npm ci
```

Start a local development server:

```bash
npm run dev
```

Build the static site:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Structure

- `src/pages/index.astro` - homepage content, layout, and styles
- `public/favicon.svg` - favicon
- `public/CNAME` - custom domain configuration
